# Список дел — PWA

Веб-приложение для управления списком дел с поддержкой офлайн-режима.

## Что сделано

- Практика 13: приложение с добавлением/удалением задач, хранение в `localStorage`, регистрация Service Worker для кэширования и офлайн-доступа
- Практика 14: добавлен `manifest.json`, иконки, мета-теги для Android и iOS, возможность установки как PWA

## Запуск
```bash
npx live-server
```
